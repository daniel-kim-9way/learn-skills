```
C1 VALUE ──→ C2 PLAN ──→ C3 BUILD ──→ C4 LAUNCH ──┐

                           C3 ────────────────────── ├──→ C6 CLAUDE CODE 심화 (도구 마스터)

```

- C5, C7: C4 LAUNCH 완료 필요 (배포된 서비스 기반)
