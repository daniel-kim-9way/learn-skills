# C3 Block 5: git + GitHub — 내 코드 저장하고 올리기

## 학습 목표
- git의 핵심 개념(커밋, 히스토리)을 이해할 수 있다
- `git init`, `git add`, `git commit` 명령어를 사용할 수 있다
- GitHub에 저장소를 만들고 코드를 push할 수 있다

## 사전 조건
- `/c3-my-skill` 완료 (나만의 스킬 생성 경험)
- GitHub 계정 (없으면 github.com에서 무료 가입)

## STOP PROTOCOL

이 스킬은 2-Phase STOP Protocol을 따릅니다.

### Phase A (Turn 1)
1. references/block5-git-github.md의 EXPLAIN 섹션을 읽고 설명한다
   - git이 무엇인지 (게임 세이브 비유)
   - 커밋 = 세이브 포인트, GitHub = 클라우드 세이브
   - 왜 개발자들이 git을 쓰는지
2. references/block5-git-github.md의 EXECUTE 섹션을 따라 실행한다
   - `git init` + `git add` + `git commit`으로 로컬 저장
   - GitHub에 저장소 생성
   - Block 4에서 만든 스킬을 push
3. **STOP** — 여기서 멈추고 학생의 응답을 기다린다. AskUserQuestion을 사용하지 않는다.
4. 멈추기 전 출력: "코드를 GitHub에 올렸습니다. '다음'을 입력하면 퀴즈로 넘어갑니다."

### Phase B (Turn 2)
1. references/block5-git-github.md의 QUIZ 섹션 퀴즈 출제
2. 학생 답변에 피드백한다 (오답 해설 포함)
3. 다음 안내:
   - "웹에서 '완료' 버튼을 눌러주세요."
   - "다음 블록: `/c3-rails-init`"
