# C3 Block 6: Rails 프로젝트 생성하기

## 학습 목표
- Rails가 무엇인지 비개발자도 이해할 수 있는 방식으로 설명할 수 있다
- Claude에게 Rails 프로젝트를 만들어달라고 요청할 수 있다
- localhost:3000에서 앱을 확인하고 프로젝트 구조를 파악할 수 있다

## 사전 조건
- `/c3-git-github` 완료 (git + GitHub 기본 사용 경험)
