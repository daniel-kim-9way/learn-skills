# C4 Block 12 (보너스): 업무 자동화 — 배운 것으로 일을 바꾸기

## 학습 목표
- C3~C4에서 배운 스킬을 업무 자동화에 적용할 수 있다
- MCP + Skill 조합으로 나만의 자동화 도구를 만들 수 있다
- 완성한 자동화 스킬을 GitHub에 공유할 수 있다

## 사전 조건
- `/c4-showcase` 완료 (C4 전 과정 완료)
- 자동화하고 싶은 업무 아이디어 1개

## STOP PROTOCOL

이 스킬은 2-Phase STOP Protocol을 따릅니다.

### Phase A (Turn 1)
1. references/block12-automation.md의 EXPLAIN 섹션을 읽고 설명한다
   - 서비스 빌딩 스킬 = 업무 자동화 스킬
   - MCP + Skills = 자동화 툴킷
   - 어떤 업무를 자동화할 수 있는가
2. references/block12-automation.md의 EXECUTE 섹션을 따라 실행한다
   - 자동화 아이디어 선정
   - Template+STUB 방식으로 자동화 스킬 제작
   - GitHub에 공유
3. **STOP** — 여기서 멈추고 학생의 응답을 기다린다. AskUserQuestion을 사용하지 않는다.
4. 멈추기 전 출력: "자동화 스킬을 완성했습니다. '다음'을 입력하면 마지막 질문으로 넘어갑니다."

### Phase B (Turn 2)
1. references/block12-automation.md의 QUIZ 섹션 성찰 질문 출제
2. 학생의 모든 답변에 진심으로 축하하고 격려한다
3. C4 완료 축하 메시지:
   - "🎉 C4 전 과정을 완료하셨습니다! 서비스를 만들고, 배포하고, 세상에 공개하고, 업무까지 자동화했습니다. 이 여정이 첫 번째 서비스의 시작입니다."
4. 다음 안내:
   - "웹에서 '완료' 버튼을 눌러주세요."
