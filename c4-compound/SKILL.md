# C4 Block 10: 학습 복리 — 지식이 쌓일수록 빨라진다

## 학습 목표
- 학습 복리의 개념을 이해한다 (지식이 지식을 부르는 효과)
- content-digest 스킬로 외부 콘텐츠를 흡수하는 방법을 익힌다
- compound 문서를 만들어 지식을 체계적으로 축적할 수 있다

## 사전 조건
- `/c4-session-wrap` 완료 (Multi-Agent 패턴 경험)

## 교수법: Quiz-First + Multi-Agent

이 블록은 Quiz-First 방식과 Multi-Agent를 결합합니다. 먼저 퀴즈로 현재 학습 방식을 점검하고, content-digest를 경험하고, Multi-Agent로 compound 문서를 만듭니다.

## STOP PROTOCOL

이 스킬은 2-Phase STOP Protocol을 따릅니다.

### Phase A (Turn 1)
1. references/block10-compound.md의 EXPLAIN 섹션을 읽고 설명한다
   - 학습 복리 개념 (복리 이자 비유)
   - content-digest란 무엇인지
   - compound 문서의 역할
2. references/block10-compound.md의 EXECUTE 섹션을 따라 실행한다
   - Pre-Quiz (학습 방식 점검)
   - content-digest 체험
   - Multi-Agent로 compound 문서 생성
3. **STOP** — 여기서 멈추고 학생의 응답을 기다린다. AskUserQuestion을 사용하지 않는다.
4. 멈추기 전 출력: "학습 복리 시스템을 구축했습니다. '다음'을 입력하면 퀴즈로 넘어갑니다."

### Phase B (Turn 2)
1. references/block10-compound.md의 QUIZ 섹션 메인 퀴즈 출제
2. 학생 답변에 피드백한다 (오답 해설 포함)
3. 다음 안내:
   - "웹에서 '완료' 버튼을 눌러주세요."
   - "다음 블록: `/c4-showcase`"
