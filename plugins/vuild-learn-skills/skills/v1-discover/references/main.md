## CONTEXT

이 스킬은 학습자가 Claude AI 도구를 처음 만나는 시점에 실행된다.
"Claude 종류가 왜 이렇게 많아?" 혼란을 해소하고, 예산에 맞는 플랜을 골라주고, API 비용 폭주를 막는 안전장치를 설정하도록 안내하는 것이 목적이다.

### Claude 4가지 도구 — 같은 두뇌, 다른 창구

| 도구 | 어디서 | 스킬 가능? | 언제 쓰나 |
|---|---|---|---|
| claude.ai | 브라우저 (웹) | ❌ | 가벼운 대화·번역·글 다듬기 |
| Claude Desktop | 데스크톱 앱 (Mac/Win) | ✅ 슬래시 스킬 | LEARN Ch.0~Ch.3 대부분 |
| Claude Code | 터미널 (CLI) | ✅ + 코드 수정 | Ch.4 BUILD 실제 코딩 |
| Claude API | 본인 코드 안에서 호출 | N/A | 본인 SaaS에 AI 기능 붙일 때 |

핵심: 지금(Ch.0~Ch.3)은 **Claude Desktop 슬래시 스킬**이 메인. Ch.4부터 Claude Code, Ch.4-13 이후 API.

### 구독 플랜 기준표

| 플랜 | 월 비용 | 대화량 | 권장 단계 |
|---|---|---|---|
| Free | $0 | 제한 (5시간마다 리셋) | Ch.0~Ch.1 맛보기 |
| Pro | $20 (약 28,000원) | Free × 5 | Ch.2~Ch.5 본격 학습 |
| Max $100 | $100 (약 14만원) | Pro × 5 | Ch.4 BUILD 집중주 |
| Max $200 | $200 (약 28만원) | Max × 2 | 전업 빌더 / 여러 프로젝트 병행 |

**단계별 권장:**
- Ch.0~Ch.1(지금) → Free 충분
- Ch.2 PLAN 시작 시 → Pro 결제 권장 (긴 대화 많아짐)
- Ch.4 BUILD 집중주 → Pro 또는 Max $100
- Ch.5 LAUNCH 후 → Pro로 내려도 됨

### API 비용 안전장치 (console.anthropic.com)

| 설정 | 권장값 | 어디서 |
|---|---|---|
| Monthly Spend Limit | $10~$30 (첫 달 보수적으로) | Settings → Limits |
| Email Alert | 50% / 80% / 100% 도달 시 메일 | 같은 페이지 Notifications |
| Hard Stop | 한도 초과 시 자동 차단 ON | 기본값 유지 |

가장 큰 위험: 코드 버그로 무한 호출 → 한도 설정 안 했으면 수십만 원 청구 가능. 미리 설정하면 절대 안 일어남.

## EXECUTE

### 인트로

다음 문장으로 시작한다:

> "안녕하세요! /v1-discover 시작합니다.
> Claude AI 도구가 4종류라서 헷갈리시죠? 몇 가지만 여쭤보면 딱 맞는 플랜을 추천해드릴게요.
> 질문 5개, 5분이면 끝납니다. 시작할게요!"

---

### Q1: 예산

> "AI 도구(Claude)에 매월 얼마까지 지출이 부담 없으세요?
> 예: '무료가 좋아요' / '2만~3만원' / '5만원 이상도 괜찮아요'"

**답변별 플랜 기준:**
- 무료 → Free 추천
- ~3만원 → Pro 추천
- 5만원+ / 속도가 중요 → Max $100 후보

---

### Q2: 학습 속도

> "이 커리큘럼을 어떤 페이스로 진행할 예정이에요?
> (a) 천천히 — 주말·틈틈이 (b) 보통 — 주 2~3일 (c) 집중적으로 — 거의 매일"

**집중형(c)이면 Max 고려**, 나머지는 Pro 이하.

---

### Q3: API 기능 계획

> "나중에 본인 서비스에 AI 기능을 붙일 계획이 있으신가요?
> (예: 고객 문의 자동 답변, 내용 자동 요약, 리뷰 분석 등)
> '아직 모르겠어요'도 괜찮아요."

**API 기능 계획 있으면** → console.anthropic.com 한도 설정 단계 필수 진행.

---

### Q4: 카드 등록 가능 여부

> "claude.ai 구독 결제가 필요할 경우, 한국 신용/체크카드 사용 가능하신가요?
> (대부분 한국 카드 정상 결제. 드물게 안 되면 PayPal 우회)"

**카드 없거나 막히면** → Free 유지 + Ch.2 시작 전 재결제 안내.

---

### Q5: 알람 메일 주소

> "API 비용 알람을 받을 이메일 주소를 알려주세요.
> (API 쓰면 비용이 생기는데, 한도 초과 알람을 미리 설정해두면 폭주 걱정 없어요.
> Anthropic 가입 메일과 달라도 됩니다.)"

---

### 추천 결과 전달

5개 답변을 종합해서 다음 형태로 정리해준다:

> "분석 결과입니다!
>
> ✅ 추천 플랜: **[Free / Pro / Max $100]**
> 이유: [예산 XX원 + 학습 속도 XX + API 기능 계획 XX 기준]
>
> 📅 업그레이드 시점: [Ch.XX 시작 전에 Pro로 올리세요]
>
> 🔒 API 안전장치: [필요/나중에] — console.anthropic.com → Settings → Limits에서 월 한도 $XX 설정 권장"

---

### API 한도 설정 안내 (API 기능 계획 있는 경우)

다음 단계를 안내한다:

1. `console.anthropic.com` 접속
2. 좌측 메뉴 **Settings → Limits**
3. **Monthly Spend Limit**: $10 입력 (나중에 서비스 성장하면 올리면 됨)
4. **Notifications**: 50%, 80%, 100% 알람 메일 켜기 (Q5에서 받은 주소로)
5. **Hard Stop** ON 확인 (기본값)

> "이거 해두면 코드 버그로 무한 호출이 생겨도 자동으로 차단돼요. 잠 잘 수 있는 안전벨트예요."

## 주의 사항

- 질문은 한 번에 하나씩. 5개를 한꺼번에 쏟지 않는다.
- 답변이 애매하면 간단히 재확인 ("혹시 매월 2~3만원 범위면 Pro 추천인데, 맞을까요?")
- API 기능 계획이 없어도 한도 설정 단계는 간략히 언급한다 ("나중에 필요하면 이렇게 하면 돼요")
- 플랜 강매 금지 — 학습자의 예산과 페이스를 존중한다
