# V1: 첫 빌드 — 자기소개 페이지 30분

## 목표
- AI와 대화해서 자기소개 HTML 파일 1개를 완성한다
- 본인 PC 브라우저에서 더블클릭으로 직접 열어본다
- (선택) Netlify Drop으로 진짜 인터넷 URL을 손에 쥐고 한 사람에게 카톡으로 공유한다

## 사전 조건
- Claude Desktop 설치 완료 (Ch.0-2)

## 진행 방식

1. `references/main.md`를 읽고 Turn 순서대로 진행한다
2. 질문을 한 번에 하나씩 던진다 (4가지를 한꺼번에 묻지 않는다)
3. 4가지 답변(이름, 소개, 잘하는 것, 디자인 톤)을 받은 후 HTML을 생성한다
4. 파일 열기 방법을 안내하고, 선택 배포(Netlify Drop)를 제안한다

## 산출물
- 파일명: `my-intro.html`
- 내용: 이름·소개·잘하는 것 3가지·연락처가 담긴 자기소개 한 페이지 (보라-핑크 그라데이션 or 선택 톤)
