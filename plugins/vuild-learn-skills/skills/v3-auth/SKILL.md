# V3: 홈페이지 + 회원가입/로그인

## 학습 목표
- 인증(회원가입/로그인)의 개념을 "집 열쇠" 비유로 이해한다
- Rails 8 내장 인증 기능을 활용해 회원가입/로그인을 구현한다
- 로그인 여부에 따라 다른 화면을 보여주는 네비게이션을 만든다

## 사전 조건
- `/v3-database` 완료 (모델과 마이그레이션 경험)
- 코드 에디터 + Claude Code 확장 프로그램에서 진행

## STOP PROTOCOL

이 스킬은 가장 긴 레슨입니다 (60분). 중간 마일스톤을 안내합니다.

### Phase A (Turn 1)
1. references/main.md의 EXPLAIN 섹션을 읽고 설명한다
   - 인증 = 집 열쇠 (회원가입=열쇠 만들기, 로그인=열쇠로 문열기)
   - Rails 8에 이미 인증 기능이 내장되어 있다
   - 홈페이지 업그레이드 계획
2. references/main.md의 EXECUTE 섹션을 따라 실행한다
   - 마일스톤 1: 홈페이지 디자인 업그레이드
   - 마일스톤 2: Rails 8 인증 기능 설정
   - 마일스톤 3: 회원가입/로그인/로그아웃 테스트
   - 마일스톤 4: 네비게이션 바 (로그인/비로그인 분기)
   - 마일스톤 5: "시작하기" 버튼 연결 + 전체 흐름 테스트
3. **STOP** — 여기서 멈추고 학생의 응답을 기다린다.
4. 멈추기 전 출력: "회원가입과 로그인이 동작합니다! '다음'을 입력하면 퀴즈로 넘어갑니다."

### Phase B (Turn 2)
1. references/main.md의 QUIZ 섹션 퀴즈 출제
2. 학생 답변에 피드백한다 (오답 해설 포함)
3. 다음 안내:
   - "웹에서 '완료' 버튼을 눌러주세요."
   - "다음 스킬: `/v3-feature-1`"
