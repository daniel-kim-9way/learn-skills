# V4: 런치 알림 준비 — OG · 한 줄 소개 · 방문자 · 런치 글

## 학습 목표
- 본인 서비스 URL을 카톡에 보냈을 때 예쁜 미리보기 카드가 뜨도록 OG(Open Graph) 4개 메타태그를 점검·추가한다
- 본인 서비스의 ‘3초 룰’ 한 줄 소개를 다듬어서 랜딩 / OG / vuild.kr checklist 3곳에 동일하게 적용한다
- 방문자 수 분석 도구(Vuild 자체 / GA4 / Plausible) 중 1개를 본인 사이트에 설치
- Disquiet · Product Hunt · 본인 SNS 3곳에 올릴 런치 글 초안을 본인 톤으로 미리 써 둔다

4. 마지막 Turn에서 OG 점검 결과 + 한 줄 소개 최종본 + 3개 런치 글 초안을 파일로 저장
