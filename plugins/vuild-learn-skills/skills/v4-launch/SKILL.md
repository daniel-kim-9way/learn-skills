# V4: 첫 사용자 5명 + 카톡 공유

## 목표
- 본인 페르소나(Ch.2 target)에 맞는 5명 후보를 선정한다
- 한 사람씩 다른 톤·다른 부탁으로 카톡 메시지 5개를 초안 작성한다
- 5명 중 최소 3명에게 첫 반응(가입·사용·한 줄 코멘트)을 받는다
- 받은 반응을 정리하고 다음 한 사이클의 우선순위 1개를 정한다

## 사전 조건
- 배포 완료 (ch5-1, 인터넷 URL 있음)
- Claude Code 터미널에서 실행

## 진행 방식

1. `references/main.md`를 읽고 5단계 코칭 흐름대로 진행한다
2. 각 단계에서 구체적인 질문을 던지고 답변을 받는다
3. 받은 답변을 바탕으로 메시지 초안·순서 등을 즉시 작성해 보여준다
4. 최종 결과를 파일로 저장한다
