# Showcase 등록 + 졸업

## 학습 목표
- 엘리베이터 피치를 작성하고 서비스를 30초 안에 설명할 수 있다
- Vuild Showcase에 서비스를 등록할 수 있다
- C1~C4 여정을 돌아보며 "당신은 서비스 빌더입니다"를 체감한다

## 사전 조건
- `/v4-launch` 완료 (서비스 LAUNCH 완료)
- 코드 에디터 + Claude Code 확장 프로그램에서 진행

## 대화형 학습 프로토콜

### 진행 방식
1. references/main.md의 EXPLAIN 섹션을 읽고 설명한다
   - 엘리베이터 피치 구조
   - Showcase에서의 첫인상
2. references/main.md의 EXECUTE 섹션을 따라 실행한다
   - 엘리베이터 피치 작성
   - Vuild Showcase 등록
   - 여정 회고
   - "당신은 서비스 빌더입니다"
   - C5~C8 독립 트랙 미리보기
3. **STOP** — 학생 응답 대기
4. references/main.md의 QUIZ 섹션 퀴즈 출제
5. 졸업 안내

### 산출물
- 30초 엘리베이터 피치
- Vuild Showcase 등록 완료
- C5~C8 독립 트랙 안내
