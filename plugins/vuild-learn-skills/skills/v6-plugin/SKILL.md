# V6: 플러그인 만들기 실습

## 학습 목표
- 본인이 일상적으로 반복하는 작업을 발견하고 플러그인 후보 1개로 정리한다
- 3가지 옵션(Claude Code 플러그인 / VS Code 확장 / Figma 플러그인)의 차이를 이해하고 본인에게 맞는 1개를 선택한다
- 선택한 옵션의 minimal 플러그인 파일을 자동 생성받고, 본인 PC에서 실제로 1회 실행해 결과를 확인한다

## 사전 조건
- Ch.5 완료 (또는 본인 SaaS 출시 후 다음 단계 학습자)
- Claude Desktop 또는 Claude Code 사용 가능
- 본인이 매주 1회 이상 반복하는 컴퓨터 작업이 있을 것 (없어도 진행 중에 같이 찾음)

## 대화형 학습 프로토콜

이 스킬은 대화형 코칭 방식을 따릅니다. 강의가 아닌 질문 → 답변 → 피드백 → 자동 코드 생성 루프로 진행합니다.

### 진행 방식
1. references/main.md를 읽고 8개 Turn의 대화 흐름에 따라 진행한다
2. 학습자의 일상 워크플로우를 분석 → 자동화 후보 도출 → 호스트 도구 결정 → 자동 코드 생성
3. 본인 PC에 등록 + 1회 실행 검증까지 함께한다
4. 산출물 파일을 작성해서 학습자에게 위치를 알려 준다

### 산출물
- 파일명: `learn-outputs/ch6-1-plugin.md`
- 내용: 선택한 호스트 도구, 플러그인 명세(이름·호출 명령·입출력), 생성된 코드/파일 위치, 본인 PC 등록 가이드, 다음 주 재호출 시 체크리스트
