## EXPLAIN

### Ch.1 마지막 — 5조각을 한 장으로 🗺️

이번 레슨은 Ch.1의 **마지막 — Reflection** 레슨이에요.

Ch.1의 5개 lesson은 각각 한 부품을 깊게 봤어요:
- **1-1 v1-webflow** — 5-layer 인프라 큰 그림 + 영어 약어 30개
- **1-2 v1-data-flow** — DB 표 + 비밀번호 해시 + OAuth
- **1-3 v1-director** — 코딩 환경 (에디터·터미널·git)
- **1-4 (lesson)** — AI 도구 + 비용 시뮬레이터
- **1-5 v1-money-flow** — 결제(PG) + API 호출 + webhook

이번 1-6은 그 5개를 **본인 SaaS 한 장**에 모아요.

왜냐하면 5개를 따로 머리에 두면 실제 만들 때 "이게 어디 들어가는 거였지?"가 돼요. 하지만 한 장으로 모으면 Ch.4 BUILD에서 "지금 만드는 부분이 이 지점이구나"가 즉시 잡혀요.

### 이번 레슨의 3가지 산출물

1. **본인 SaaS의 종합 인프라 지도 한 장** — 캡쳐해서 책상 옆에 두면 Ch.4 BUILD가 든든해요.
2. **자가진단 체크리스트 (6항목)** — Ch.2 진입 준비도 확인.
3. **한 줄 회고** — Ch.1 들어오기 전과 지금 본인 머릿속이 어떻게 달라졌는지.
