# V1: AI와 30분 만에 서비스 만들어보기

## 학습 목표
- AI와 협업하여 실제 동작하는 웹페이지를 만든다
- "코드 한 줄 없이 만들 수 있다"는 AHA 모먼트를 경험한다
- 수정 요청 → 즉시 반영의 피드백 루프를 체감한다

## 사전 조건
- v1-director 완료
- 웹 브라우저 사용 가능
- Claude Code Desktop에서 진행

## 대화형 학습 프로토콜

이 스킬은 대화형 코칭 방식을 따릅니다. 강의가 아닌 질문-답변-피드백 루프로 진행합니다.

### 진행 방식
1. references/main.md를 읽고 대화 흐름에 따라 진행한다
2. 학생의 선택에 따라 실제 HTML 파일을 생성한다
3. 수정 요청을 받아 실시간으로 파일을 변경한다
4. 최종 산출물을 파일로 저장한다

### 산출물
- 파일명: `first-build/index.html`
- 내용: 학생이 AI와 함께 만든 첫 번째 웹페이지 (HTML + inline CSS/JS)
