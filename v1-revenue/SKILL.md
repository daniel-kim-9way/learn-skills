# V1: 돈이 되는 구조, 간단히 보기

## 학습 목표
- 3가지 기본 수익 모델(구독/건당/프리미엄)을 이해한다
- 역산(Reverse) 계산으로 필요한 사용자 수를 파악한다
- 현실적인 수익 스케치를 작성한다

## 사전 조건
- v1-naming 완료 (my-service-identity.md 파일 필요)
- Claude Code Desktop에서 진행

## 대화형 학습 프로토콜

이 스킬은 대화형 코칭 방식을 따릅니다. 강의가 아닌 질문-답변-피드백 루프로 진행합니다.

### 진행 방식
1. references/main.md를 읽고 대화 흐름에 따라 진행한다
2. 학생의 서비스에 맞는 수익 모델을 함께 탐색한다
3. 역산 계산으로 현실감을 부여한다
4. 최종 산출물을 파일로 저장한다

### 산출물
- 파일명: `my-revenue-sketch.md`
- 내용: 선택한 수익 모델, 역산 계산 결과, 현실성 점검 메모
