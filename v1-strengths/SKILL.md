# V1: 내가 잘하는 것 x 관심 있는 것

## 학습 목표
- 자신의 강점과 관심사를 3가지 각도에서 탐색한다
- 강점과 관심사의 교차점을 발견한다
- 서비스 아이디어의 씨앗을 심는다

## 사전 조건
- v1-firstbuild 완료
- Claude Code Desktop에서 진행

## 대화형 학습 프로토콜

이 스킬은 대화형 코칭 방식을 따릅니다. 강의가 아닌 질문-답변-피드백 루프로 진행합니다.

### 진행 방식
1. references/main.md를 읽고 대화 흐름에 따라 진행한다
2. 3가지 질문으로 삼각측량(Triangulation) 방식으로 강점을 탐색한다
3. AI가 교차점을 분석하고 서비스 아이디어 방향을 제안한다
4. 최종 산출물을 파일로 저장한다
