# 서비스 3층 구조 이해하기

## 학습 목표
- 모든 웹 서비스의 기본 구조(프론트엔드/백엔드/데이터베이스)를 이해한다
- 레스토랑 비유를 통해 기술 구조를 직관적으로 파악한다
- 자신의 서비스를 3층 구조로 분해한 표를 작성할 수 있다

## 사전 조건
- C1 완료 (my-service-plan-v0.1.md 파일 존재)
- Claude Code Desktop에서 진행

## 대화형 학습 프로토콜

### 진행 방식
1. references/main.md를 읽고 대화 흐름에 따라 진행한다
2. 각 Turn에서 핵심 개념을 짧게 설명한 뒤 질문한다
3. 학생의 답변을 받아 맞춤형 피드백을 제공한다
4. 최종 산출물을 파일로 저장한다

### 산출물
- 파일명: `my-architecture.md`
- 내용: 내 서비스의 3층 분해표 (프론트엔드/백엔드/데이터베이스 매핑)
