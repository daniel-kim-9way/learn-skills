# 블루프린트 생성 + 확인

## 학습 목표
- AI가 생성한 기술 블루프린트를 읽고 핵심 결정을 이해할 수 있다
- 건물주 체크리스트로 코드 없이 설계를 검증하는 방법을 배운다
- 서비스 디렉터로서 블루프린트에 승인(sign-off)을 할 수 있다

## 사전 조건
- `/v2-user-flow` 완료 (my-user-flow.md 존재)
- Claude Code Desktop에서 진행

## 대화형 학습 프로토콜

### 진행 방식
1. references/main.md를 읽고 대화 흐름에 따라 진행한다
2. AI가 학생의 서비스에 맞는 기술 블루프린트를 생성한다
3. 건물주 체크리스트로 핵심 포인트만 함께 확인한다
4. 학생의 승인을 받아 최종 블루프린트를 파일로 저장한다

### 산출물
- 파일명: `my-blueprint.md`
- 내용: 서비스 디렉터 승인 완료된 기술 블루프린트
