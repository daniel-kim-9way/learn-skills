# 내 서비스 유형 결정하기

## 학습 목표
- 4가지 서비스 유형(웹앱/챗봇/자동화/랜딩페이지)의 차이를 이해한다
- 의사결정 트리를 통해 자신의 서비스에 맞는 유형을 선택한다
- 선택의 이유를 논리적으로 설명할 수 있다

## 사전 조건
- `/v2-architecture` 완료 (3층 구조 이해 + my-architecture.md 존재)
- Claude Code Desktop에서 진행

## 대화형 학습 프로토콜

### 진행 방식
1. references/main.md를 읽고 대화 흐름에 따라 진행한다
2. 4가지 질문으로 구성된 의사결정 트리를 순서대로 진행한다
3. 학생의 답변을 바탕으로 유형을 추천하고 함께 확정한다
4. 최종 산출물을 파일로 저장한다

### 산출물
- 파일명: `my-service-type.md`
- 내용: 선택한 서비스 유형 + 선택 이유 + 의사결정 과정
