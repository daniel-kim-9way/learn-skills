# 사용자 흐름 그리기 + 검증

## 학습 목표
- Must 기능의 사용자 흐름을 텍스트 다이어그램으로 그릴 수 있다
- 3-클릭 룰, 막다른 길, 에러 복구 개념을 이해하고 적용한다
- 까다로운 사용자 역할극을 통해 흐름의 문제점을 발견하고 수정한다

## 사전 조건
- `/v2-priority` 완료 (my-priority-list.md 존재)
- Claude Code Desktop에서 진행

## 대화형 학습 프로토콜

### 진행 방식
1. references/main.md를 읽고 대화 흐름에 따라 진행한다
2. Must 기능 Top 3의 사용자 흐름을 텍스트 다이어그램으로 그린다
3. AI가 까다로운 사용자 역할을 하며 흐름을 테스트한다
4. 3가지 규칙(막다른 길 없음, 3-클릭 룰, 에러 복구)으로 검증 후 파일로 저장한다

### 산출물
- 파일명: `my-user-flow.md`
- 내용: 검증 완료된 사용자 흐름도 (Top 3 Must 기능)
