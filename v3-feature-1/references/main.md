## EXPLAIN

### "영화 감독" 마인드셋

영화 감독은 직접 연기하지 않습니다. 대본을 쓰고, 배우에게 지시하고, 결과를 확인하고, 피드백합니다.

여러분도 마찬가지입니다:
- **직접 코딩하지 않습니다** (그건 Claude의 일)
- **무엇을 만들지 계획합니다** (감독의 비전)
- **Claude에게 지시합니다** (배우에게 연기 지시)
- **결과를 확인합니다** (촬영본 리뷰)
- **피드백을 줍니다** ("이 부분을 다르게 해줘")

이것이 AI 시대의 서비스 빌딩 방식입니다.

### 큰 기능을 작은 조각으로 나누기

"독서 모임에서 책을 관리하는 기능"을 한 번에 만들면 너무 큽니다. 이렇게 나눕니다:

1. 책 목록 페이지 (저장된 책들을 보여주기)
2. 책 추가 페이지 (새 책을 등록하는 폼)
3. 책 상세 페이지 (책 하나의 자세한 정보)
4. 책 수정 기능
5. 책 삭제 기능
6. 디자인 다듬기

작은 조각으로 나누면 좋은 점:
- 한 조각이 10분 안에 끝남
- 중간에 확인할 수 있음
- 문제가 생겨도 어디서 생겼는지 알기 쉬움

### 구현 패턴: 계획 → 조각내기 → 지시 → 확인 → 반복 → 세이브

이 패턴이 C3 전체에서 가장 중요합니다. 외울 필요는 없지만, 이번 레슨에서 이 패턴을 몸에 익힙니다:

1. **계획**: C2 기획서에서 만들 기능을 확인한다
2. **조각내기**: 기능을 작은 단위로 나눈다
3. **지시**: Claude에게 첫 번째 조각을 요청한다
4. **확인**: 브라우저에서 결과를 확인한다
5. **반복**: 다음 조각을 요청한다 (문제가 있으면 수정 요청)
6. **세이브**: 전체가 완성되면 git commit

## EXECUTE

### 단계 1: C2 핵심 기능 확인

Claude에게 이렇게 요청하세요:
```
CLAUDE.md를 읽고, 내 서비스의 핵심 기능 1번이 뭔지 알려줘.
이 기능을 구현하려면 어떤 작은 조각들이 필요한지 목록으로 만들어줘.
```

Claude가 기능을 분석하고 조각 목록을 만들어줍니다. 예를 들어:

"독서 목록 관리" 기능이라면:
1. 책 목록 페이지
2. 책 추가 폼
3. 책 저장 기능
4. 책 상세 보기
5. 책 수정/삭제
6. 디자인 정리

### 단계 2: 첫 번째 조각 구현

Claude에게 이렇게 요청하세요:
```
그러면 첫 번째 조각인 [조각명]부터 만들어줘.
로그인한 사용자만 볼 수 있어야 해.
```

Claude가 필요한 파일들(모델, 컨트롤러, 뷰, 라우트)을 만듭니다.

브라우저에서 확인합니다: `http://localhost:3000`

### 단계 3: 두 번째 조각 구현

Claude에게 이렇게 요청하세요:
```
좋아! 다음 조각인 [조각명]을 만들어줘.
[구체적인 요구사항도 함께 전달]
```

브라우저에서 확인합니다. 문제가 있으면:
```
[문제 설명]. 수정해줘.
```

### 단계 4: 나머지 조각들 반복

같은 패턴으로 나머지 조각들을 진행합니다:
```
다음 조각: [조각명].
[추가 요구사항].
```

각 조각마다 반드시 브라우저에서 확인합니다.

### 단계 5: 디자인 다듬기

기본 기능이 다 되면 디자인을 정리합니다:

Claude에게 이렇게 요청하세요:
```
지금 만든 [기능명] 페이지들의 디자인을 깔끔하게 정리해줘.
- 목록은 카드 형태로
- 버튼은 홈페이지와 통일된 스타일로
- 빈 목록일 때 안내 메시지 표시
```

### 단계 6: 전체 흐름 테스트

처음부터 끝까지 사용자 입장에서 테스트합니다:

체크리스트:
- [ ] 로그인한 상태에서 기능 페이지에 접근
- [ ] 새 데이터 추가
- [ ] 데이터 목록 확인
- [ ] 데이터 상세 보기
- [ ] 데이터 수정
- [ ] 데이터 삭제
- [ ] 비로그인 상태에서는 접근 불가

문제가 있으면:
```
[문제 설명]. 해결해줘.
```

### 단계 7: Git 세이브 + CLAUDE.md 업데이트

Claude에게 이렇게 요청하세요:
```
변경사항을 Git에 저장해줘. 메시지는 "핵심 기능 1: [기능명] 구현 완료"로.
CLAUDE.md의 현재 상태도 "v3-feature-1 완료. [기능명] 구현."으로 업데이트해줘.
```
