# Tailwind CSS로 디자인 입히기

## 학습 목표
- 디자인이 "예쁨"이 아니라 "신뢰"를 만드는 수단임을 이해한다
- Before/After 비교를 통해 디자인의 힘을 체감한다
- Claude에게 디자인 개선을 요청하고 반복적으로 다듬는 방법을 익힌다

## 사전 조건
- `/v4-warmup` 완료 (서비스 실행 및 컨텍스트 회복)
- 코드 에디터 + Claude Code 확장 프로그램에서 진행

## 대화형 학습 프로토콜

### 진행 방식
1. references/main.md의 EXPLAIN 섹션을 읽고 설명한다
   - 디자인 = 신뢰 (0.05초 판단)
   - "디자인은 신뢰다" 메시지 전달
   - Tailwind CSS가 무엇인지
2. references/main.md의 EXECUTE 섹션을 따라 실행한다
   - Before 스크린샷 촬영
   - 3가지 프롬프트로 디자인 개선 (레이아웃/카드/테마)
   - 반복 다듬기
   - After 비교
3. **STOP** — 학생 응답 대기
4. references/main.md의 QUIZ 섹션 퀴즈 출제
5. 다음 안내: `/v4-landing`

### 산출물
- Before/After 스크린샷 비교
- Tailwind CSS 적용된 서비스 화면
