# 보안 체크리스트

## 학습 목표
- 보안이 "무서운 것"이 아니라 "내 사용자를 보호하는 방패"임을 이해한다
- 10개 보안 체크리스트를 실행하고 문제를 해결할 수 있다
- 배포 전 보안 점검이 필수인 이유를 안다

## 사전 조건
- `/v4-landing` 완료 (서비스 기본 기능 완성)
- 코드 에디터 + Claude Code 확장 프로그램에서 진행

## 대화형 학습 프로토콜

### 진행 방식
1. references/main.md의 EXPLAIN 섹션을 읽고 설명한다
   - "내 사용자를 보호하는 방패" (empowering, not scary)
   - 주요 보안 위협을 일상 비유로 설명
2. references/main.md의 EXECUTE 섹션을 따라 실행한다
   - 10개 체크리스트 자동 점검
   - 문제 항목 즉시 수정
   - 보안 sign-off
3. **STOP** — 학생 응답 대기
4. references/main.md의 QUIZ 섹션 퀴즈 출제
5. 다음 안내: `/v4-deploy`

**중요: 이 스킬은 반드시 배포(/v4-deploy) 전에 완료해야 합니다.**

### 산출물
- 10개 보안 체크리스트 전체 통과
- 보안 sign-off 완료
