# C5 Block 5: 포트폴리오로 활용하기

## 학습 목표
- STAR-M 프레임워크로 프로젝트를 정리할 수 있다
- 임팩트 있는 프로젝트 설명을 작성할 수 있다
- 서비스 빌딩 경험을 커리어에 활용하는 방법을 안다

## 사전 조건
- C4 LAUNCH 완료 (배포된 서비스 보유)
- 코드 에디터 + Claude Code 확장 프로그램 설치
- `/v5-saas-ops` 완료 (SaaS 운영 전략)

## STOP PROTOCOL

이 스킬은 2-Phase STOP Protocol을 따릅니다.

### Phase A (Turn 1)
1. references/main.md의 EXPLAIN 섹션을 읽고 설명한다
   - STAR-M 프레임워크
   - Before/After 숫자의 힘
   - 커리어 활용 전략
2. references/main.md의 EXECUTE 섹션을 따라 실행한다
   - STAR-M으로 프로젝트 정리
   - Before/After 숫자 정리
   - 포트폴리오 문서 작성
   - my-portfolio.md 작성
3. **STOP** — 여기서 멈추고 학생의 응답을 기다린다. AskUserQuestion을 사용하지 않는다.
4. 멈추기 전 출력: "포트폴리오 작성을 완료했습니다. '다음'을 입력하면 퀴즈로 넘어갑니다."

### Phase B (Turn 2)
1. references/main.md의 QUIZ 섹션 퀴즈 출제
2. 학생 답변에 피드백한다 (모든 답변이 유효, 경험 격려)
3. 다음 안내:
   - "웹에서 '완료' 버튼을 눌러주세요."
   - "C5 '서비스 성장시키기'를 모두 완료했습니다! 축하합니다!"

## 워밍업 태스크 (재개 시)

스킬 시작 시 프로젝트 디렉토리가 존재하면:
1. 프로젝트 상태를 기능 중심으로 요약 (기술 용어 없이)
2. 작은 수정 태스크 1개 제시 (버튼 색 변경, 텍스트 수정 등)
3. 성공 확인 후 본 학습 시작

프로젝트가 없으면 워밍업을 건너뛴다.
