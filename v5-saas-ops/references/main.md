## EXPLAIN

### Revenue Stacking: 수익을 층층이 쌓는 법

"월 100만원"이라고 하면 거창해 보입니다. 하지만 분해해보면 생각보다 현실적입니다.

**핵심 원리: 한 가지 수익원에 의존하지 않는다.**

수익을 하나의 큰 덩어리로 만들려고 하면 어렵습니다. 대신 여러 층으로 쌓으면 훨씬 안정적입니다.

```
[Revenue Stacking 구조]

┌─────────────────────────────────────┐
│  4층: 부가 서비스                    │  월 15~30만원
│  (1:1 컨설팅, 커스텀 개발)           │  소수 고객, 높은 단가
├─────────────────────────────────────┤
│  3층: 일회성 판매                    │  월 10~20만원
│  (템플릿, 가이드, 강의)              │  한 번 만들면 반복 판매
├─────────────────────────────────────┤
│  2층: 프리미엄 구독                  │  월 25~40만원
│  (월 29,900원 x 10~15명)            │  핵심 수익원
├─────────────────────────────────────┤
│  1층: 기본 구독                      │  월 20~30만원
│  (월 9,900원 x 20~30명)             │  진입 장벽 낮은 기본 플랜
├─────────────────────────────────────┤
│  지하: 무료 플랜 (Free)              │  수익 0원
│  (기본 기능 제공)                    │  신규 사용자 유입 통로
└─────────────────────────────────────┘
```

**구독 + 부가서비스 조합 예시:**

| 서비스 유형 | 구독 (월 반복) | 부가서비스 (일회성) |
|-------------|---------------|-------------------|
| 재고관리 SaaS | 월 9,900~29,900원 | 초기 데이터 세팅 대행 (5만원) |
| 학습 플래너 | 월 4,900~14,900원 | 맞춤 학습 계획서 작성 (3만원) |
| 식단 추천 | 월 9,900~19,900원 | 영양사 1:1 상담 (2만원) |
| 포트폴리오 빌더 | 월 9,900~29,900원 | 이력서 첨삭 서비스 (5만원) |

> **꿀팁**: 부가서비스는 "내가 직접 해주는 서비스"입니다. 시간이 들지만 단가가 높고, 고객의 진짜 니즈를 파악하는 최고의 방법입니다. 부가서비스에서 반복되는 요청이 있으면, 그걸 자동화해서 새로운 기능으로 만들면 됩니다.

### 3-Tier 가격 전략: 앵커링 효과

가격표를 보여주는 방법 하나로 매출이 달라집니다. 심리학의 **앵커링 효과**를 활용합니다.

```
[가격표 디자인 — 앵커링 적용]

┌──────────┐  ┌──────────────┐  ┌──────────┐
│  Basic   │  │  ⭐ Pro       │  │ Premium  │
│          │  │  (인기!)      │  │          │
│ 월 9,900 │  │ 월 29,900    │  │ 월 99,000│
│          │  │              │  │          │
│ · 기본기능│  │ · 기본 전부  │  │ · Pro 전부│
│ · 3건/월 │  │ · 무제한     │  │ · 전담 지원│
│          │  │ · AI 기능    │  │ · API 접근│
│          │  │ · 우선 지원  │  │ · 커스텀  │
│          │  │              │  │          │
│ [선택]   │  │ [시작하기]   │  │ [문의]   │
└──────────┘  └──────────────┘  └──────────┘
```

**앵커링이 작동하는 원리:**

1. 오른쪽의 99,000원을 먼저 봅니다 (비싸다!)
2. 가운데 29,900원이 "합리적"으로 느껴집니다
3. 왼쪽 9,900원은 "기능이 부족해 보입니다"
4. 결과: **대부분 가운데(Pro)를 선택합니다**

이것이 **앵커링 효과**입니다. 처음 본 높은 가격이 기준점(앵커)이 되어, 다른 가격이 상대적으로 저렴하게 느껴집니다.

**가격 설정 공식:**

| 플랜 | 가격 기준 | 목적 |
|------|-----------|------|
| Basic | 부담 없는 가격 | 진입 장벽 낮추기, 사용자 확보 |
| Pro | Basic의 2~3배 | **핵심 수익원**, 가장 많이 팔리게 설계 |
| Premium | Pro의 3~5배 | 앵커 역할, 소수의 고가 고객 확보 |

> **꿀팁**: 연간 결제 할인도 강력합니다. "월 29,900원 → 연 299,000원(월 24,900원, 17% 할인)"으로 제시하면, 한 번에 1년치 매출을 확보하면서 이탈도 방지할 수 있습니다.

### Churn Prevention: 이탈 신호 감지하고 막기

SaaS에서 **이탈률(Churn Rate)**은 가장 무서운 숫자입니다.

왜 무서운지 계산해봅시다:

```
[이탈률의 복리 효과]

매달 10% 이탈하는 서비스 (시작: 100명)
- 1개월 후: 90명
- 3개월 후: 73명
- 6개월 후: 53명
- 12개월 후: 28명 😱

매달 5% 이탈하는 서비스 (시작: 100명)  
- 1개월 후: 95명
- 3개월 후: 86명
- 6개월 후: 74명
- 12개월 후: 54명

차이: 이탈률 5% 차이가 1년 뒤 2배 차이!
```

**이탈 신호 감지하기 — 이런 행동이 보이면 빨간불:**

| 이탈 신호 | 의미 | 대응 |
|-----------|------|------|
| 7일 이상 미접속 | 서비스를 잊어가는 중 | 가치 리마인드 이메일 |
| 핵심 기능 미사용 | 가치를 못 느끼는 중 | "이 기능 써보셨나요?" 넛지 |
| 지원 문의 후 해결 안 됨 | 불만이 쌓이는 중 | 빠른 후속 대응 |
| 결제 실패 후 재시도 안 함 | 이탈 결심 | 결제 실패 안내 + 혜택 제안 |
| 데이터 다운로드/내보내기 | 떠날 준비 중 | 해지 사유 인터뷰 |

**리텐션 이메일 시퀀스:**

```
[가입 후 자동 이메일 시퀀스]

Day 0: 환영 이메일
  "가입을 환영합니다! 시작 가이드를 확인해보세요."

Day 1: 첫 경험 유도
  "아직 [핵심 기능]을 안 써보셨네요. 3분이면 됩니다!"

Day 3: 가치 전달
  "다른 사용자들은 이렇게 활용하고 있어요."

Day 7: 성과 리마인드
  "가입 첫 주! 지금까지 [00]을 달성하셨어요."

Day 14: 피드백 요청
  "서비스 경험은 어떠세요? 의견을 들려주세요."

Day 30: 업그레이드 안내 (무료 사용자)
  "Pro 플랜에서는 이런 것도 가능해요. 7일 무료 체험!"
```

> **막혔을 때**: "이메일 자동화 시스템을 만들기 어려워요." 처음에는 자동화 없이 직접 보내도 됩니다. 사용자가 30명 이하일 때는 직접 카카오톡이나 이메일로 연락하는 게 오히려 더 효과적입니다. "대표가 직접 연락해주다니!"라는 감동을 줄 수 있습니다.

### 한국 인디해커 실제 사례

"그런 게 진짜 되나요?"라고 생각하실 수 있습니다. 한국에서 실제로 1인(또는 소규모)으로 수익을 내고 있는 사례를 소개합니다.

**사례 1: 노션 템플릿 판매 (월 50~200만원)**

> "처음엔 제가 쓸 노션 템플릿을 만들었어요. 블로그에 올렸더니 다운로드 요청이 와서, 유료로 전환했습니다. 첫 달 매출 8만원이었는데, 6개월 뒤 월 100만원을 넘었어요."

- 수익 구조: 템플릿 판매 (9,900~39,900원) + 커스터마이징 대행 (10만원~)
- 핵심: 본인이 실제로 쓰는 템플릿 → 품질 보장

**사례 2: B2B SaaS - 소규모 학원 관리 (월 100~300만원)**

> "학원 강사 출신이라 학원 관리의 불편함을 잘 알았어요. 엑셀 대신 쓸 수 있는 간단한 웹 서비스를 만들어서, 아는 학원장님부터 시작했습니다."

- 수익 구조: 월 29,900원 구독 x 30~50개 학원
- 핵심: 본인이 업계 전문가 → 고객 니즈 정확히 파악

**사례 3: 콘텐츠 큐레이션 뉴스레터 + SaaS (월 80~150만원)**

> "매일 아침 IT 뉴스를 정리해서 보내는 뉴스레터를 시작했어요. 구독자가 모이니까 광고 수익이 생기고, 이걸 자동화하는 도구를 SaaS로 만들어서 다른 뉴스레터 운영자에게 팔기 시작했습니다."

- 수익 구조: 뉴스레터 광고 (30~50만원) + SaaS 구독 (50~100만원)
- 핵심: 콘텐츠로 신뢰 구축 → SaaS로 수익화

**공통점:**
1. 본인이 겪는 문제를 해결하는 서비스
2. 무리하지 않고 작게 시작
3. 소수의 열성 사용자 → 확장
4. 구독 + 부가서비스 조합

### 월 100만원 달성 로드맵

현실적인 3단계 로드맵입니다:

```
[1~3개월차: 씨앗 단계] — 목표: 월 10만원

할 일:
- 무료 사용자 20~30명 확보
- 피드백 수집 → 핵심 기능 개선
- 유료 플랜 1개 출시 (월 9,900원)
- 첫 유료 고객 5~10명

핵심 지표: 무료→유료 전환율 20% 이상


[4~6개월차: 새싹 단계] — 목표: 월 30~50만원

할 일:
- 3-Tier 가격 도입 (Basic/Pro/Premium)
- 부가서비스 1개 추가 (컨설팅, 대행 등)
- 이탈 방지 이메일 시퀀스 가동
- 월간 개선 루틴 정착

핵심 지표: 이탈률 10% 이하


[7~12개월차: 성장 단계] — 목표: 월 100만원

할 일:
- 연간 결제 도입 (20% 할인)
- 자동화 (이메일, 결제, 알림)
- SEO/콘텐츠로 자연 유입 증가
- 고객 추천 프로그램

핵심 지표: MRR(월 반복 매출) 100만원
```

> **꿀팁**: "월 100만원"보다 중요한 건 **"첫 유료 고객 1명"**입니다. 전혀 모르는 사람이 내 서비스에 돈을 내는 순간, 모든 게 바뀝니다. 그 1명에게 최선을 다하세요. 그 1명이 10명을 데려옵니다.

### 자동화로 운영 효율 높이기

1인 운영자가 모든 걸 수동으로 하면 24시간이 모자랍니다. 반복되는 일은 자동화합니다.

| 작업 | 수동일 때 | 자동화 후 |
|------|----------|----------|
| 환영 이메일 | 가입할 때마다 직접 발송 | Action Mailer + Solid Queue 자동 발송 |
| 결제 알림 | 결제일마다 확인 | 웹훅으로 자동 알림 |
| 에러 감지 | 사용자가 알려줄 때까지 모름 | 에러 발생 시 즉시 슬랙 알림 |
| 운영 리포트 | 매주 수동으로 데이터 정리 | 주간 자동 리포트 이메일 |

> **막혔을 때**: "자동화가 어려워요." 처음부터 전부 자동화할 필요 없습니다. 규칙은 이것입니다: **같은 일을 3번 이상 수동으로 했으면, 그때 자동화하세요.** 처음에는 수동으로 해도 괜찮습니다. 오히려 수동으로 하면서 프로세스를 제대로 이해한 뒤에 자동화하는 게 더 좋은 결과를 만듭니다.
