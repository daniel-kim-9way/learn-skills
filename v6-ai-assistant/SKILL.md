# V6: 개인 AI 비서 만들기 실습 (RAG)

## 학습 목표
- RAG(Retrieval-Augmented Generation)가 정확히 뭐고 일반 ChatGPT/Claude와 어떻게 다른지 이해한다
- 본인 자료의 양과 형태에 맞춰 두 가지 모드(긴 컨텍스트 vs Vector DB)를 구분하고 1개를 결정한다
- 본인 노트·문서·블로그 100~1000건 기반 AI 비서를 1개 띄우고, 본인이 답을 아는 5개 질문으로 정확도 검증까지 완료한다

## 사전 조건
- Ch.5 완료 또는 본인이 오래 쌓은 자료(100건+)가 있는 사람
- Anthropic API 키 (v6-chatbot에서 발급한 것 재사용 가능)
- 본인 자료가 정리된 형태 또는 정리 가능한 상태 (Notion/Google Docs/마크다운 폴더/PDF 등)

## 대화형 학습 프로토콜

이 스킬은 대화형 코칭으로 진행합니다. 본인 자료의 위치·양·형태를 같이 파악하고, 모드(긴 컨텍스트 vs Vector DB)를 결정하고, 추출 → 임베딩 → 질문 인터페이스 → 정확도 검증까지 함께해요.

### 진행 방식
1. references/main.md를 읽고 8개 Turn의 대화 흐름을 따라간다
2. 학습자 자료 파악 → RAG 개념 → 질문 정의 → 모드 결정 → 셋업 → 자료 주입 → 첫 질문 테스트 → 정확도 검증
3. Vector DB(ChromaDB) 모드는 셋업이 길어지므로, 100건 미만이면 긴 컨텍스트 모드로 빠르게 가도록 안내한다
4. 산출물 파일을 작성해서 자료 위치, 모드, 코드 위치, 정확도 결과까지 정리한다
