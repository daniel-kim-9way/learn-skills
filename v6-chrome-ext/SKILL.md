# V6: 크롬 익스텐션 만들기 실습

## 학습 목표
- 크롬 익스텐션이 정확히 뭐고 `manifest.json` 이 무슨 역할인지 이해한다
- 본인 브라우저 워크플로우의 반복 작업 1개를 발견하고 익스텐션 1개로 자동화한다
- 본인 크롬에 `chrome://extensions` 사이드로드 → 실제 페이지에서 1회 동작 확인까지 완료한다

## 사전 조건
- Ch.5 완료 또는 본인이 매일 브라우저에서 일하는 사람
- Chrome 또는 Chromium 기반 브라우저(Edge, Brave) 설치
- (Claude API 호출 기능을 쓸 경우) Anthropic API 키 — v6-chatbot에서 발급한 것 재사용 가능

## 대화형 학습 프로토콜

이 스킬은 대화형 코칭으로 진행합니다. 본인 워크플로우에서 자동화할 작업을 같이 발견하고, 익스텐션 4개 능력 중 어디에 해당하는지 정의하고, manifest + popup 코드를 자동 생성해서 본인 크롬에 사이드로드까지 함께해요.

### 진행 방식
1. references/main.md를 읽고 8개 Turn의 대화 흐름을 따라간다
2. 학습자의 브라우저 작업 → 익스텐션 능력(4가지) 분류 → 권한 결정 → 4개 파일 자동 생성 → 사이드로드 가이드 → 실제 페이지 테스트
3. API 키 보안(공개 익스텐션은 본인 백엔드 거쳐야 함) 경고를 명확히 한다
4. 산출물 파일을 작성해서 폴더 위치, 호출 방법, 친구에게 공유 방법까지 정리한다
