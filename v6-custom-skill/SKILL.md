# V6: 나만의 스킬 만들기

## 학습 목표
- Template + STUB 패턴으로 스킬의 구조를 이해한다
- 반복적인 작업을 스킬로 자동화할 수 있다
- 직접 스킬을 만들고 테스트하고 개선하는 과정을 경험한다

## 사전 조건
- C3 BUILD 완료 (Claude Code 사용 경험)
- 코드 에디터 + Claude Code 확장 프로그램 설치

## 소요 시간
- 약 40분

## 대화형 학습 프로토콜

이 스킬은 대화형 코칭 방식을 따릅니다. 실제 반복 작업을 스킬로 변환하는 과정을 함께 진행합니다.

### 진행 방식
1. references/main.md를 읽고 대화 흐름에 따라 진행한다
2. Template + STUB 패턴을 설명한다
3. 학생이 반복 작업을 선택하고 함께 STUB을 채운다
4. 스킬을 테스트하고 반복 개선한다

### 산출물
- 커스텀 스킬 파일: `.claude/skills/` 디렉토리에 저장
