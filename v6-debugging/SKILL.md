# V6: 디버깅 심화

## 학습 목표
- 에러를 유형별로 분류하고 접근 전략을 세울 수 있다
- 스택 트레이스를 읽고 문제의 원인을 추적할 수 있다
- git bisect 등 체계적인 디버깅 도구를 활용할 수 있다
- 버그를 예방하는 습관을 만든다

## 사전 조건
- C3 BUILD 완료 (Claude Code 사용 경험)
- 코드 에디터 + Claude Code 확장 프로그램 설치

## 소요 시간
- 약 30분

## 대화형 학습 프로토콜

이 스킬은 대화형 코칭 방식을 따릅니다. 에러 유형을 분류하고 체계적 접근법을 연습합니다.

### 진행 방식
1. references/main.md를 읽고 대화 흐름에 따라 진행한다
2. 에러 유형별 분류와 접근 전략을 설명한다
3. 스택 트레이스 읽기와 git bisect를 연습한다
4. 나만의 디버깅 플레이북을 작성한다

- 파일명: `my-debugging-playbook.md`
