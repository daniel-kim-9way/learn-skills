# V6: MCP — 외부 도구와 AI 연결하기

## 학습 목표
- MCP(Model Context Protocol)가 무엇인지 비유로 이해한다
- 하나의 외부 도구(Slack, Notion 등)를 Claude Code에 연결할 수 있다
- MCP가 유용한 상황과 그렇지 않은 상황을 구분할 수 있다

## 사전 조건
- C3 BUILD 완료 (Claude Code 사용 경험)
- 코드 에디터 + Claude Code 확장 프로그램 설치

## 소요 시간
- 약 30분

## 대화형 학습 프로토콜

이 스킬은 대화형 코칭 방식을 따릅니다. 비유로 개념을 잡고, 직접 연결하며 체험합니다.

### 진행 방식
1. references/main.md를 읽고 대화 흐름에 따라 진행한다
2. 레스토랑 배달 비유로 MCP 개념을 설명한다
3. 하나의 도구를 실제로 연결하고 테스트한다
4. MCP가 유용한 경우와 불필요한 경우를 정리한다

### 산출물
- 파일명: `my-mcp-setup.md`
- 내용: 연결한 도구 정보, 설정 과정, 활용 아이디어
